# async-api-fast-dio-luizalabs

## O código consiste em um microsservico assíncrono desenvolvido com FastAPI e utilizando o cliente HTTP assíncrono httpx para realizar requisições a serviços externos. A API é projetada para ser eficiente e escalável. O servico user, e responsável pelo cadastro e demais requisições do usuário, assim como o login e envio do token de autenticação, com o RabbitMq, para se ter acesso ao serviço que gerencia a conta do cliente, account service.
